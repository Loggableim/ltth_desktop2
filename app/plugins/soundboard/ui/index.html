OBS Animation Overlay

Add this URL as a Browser Source in OBS to display MP4/GIF animations when gifts are received or events occur. Recommended size: 1920x1080, transparent background.

How to use: In OBS, add a Browser Source → paste this URL → set width to 1920, height to 1080 → check "Transparent Background". Animations will play automatically when configured gifts are received.

Playback Settings

Overlap: All sounds play immediately.
Queue All: All sounds play one after another.
Queue Per Gift: Same gift types wait, different gifts can play simultaneously.
